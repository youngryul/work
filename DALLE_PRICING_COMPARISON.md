# DALL-E 2 vs DALL-E 3 가격 비교

## 이미지 생성 비용 (2024년 기준)

### DALL-E 2
- **1024×1024 해상도**: **$0.02** (약 **26원**) per image
- 더 저렴하지만 품질이 상대적으로 낮음

### DALL-E 3
- **1024×1024 해상도 (Standard)**: **$0.04** (약 **52원**) per image
- **1024×1024 해상도 (HD)**: **$0.08** (약 **104원**) per image
- DALL-E 2보다 **2배 비싸지만** 훨씬 높은 품질 제공

## 가격 차이 요약

| 모델 | 1024×1024 (Standard) | 1024×1024 (HD) | 품질 |
|------|---------------------|----------------|------|
| DALL-E 2 | $0.02 (약 26원) | - | 보통 |
| DALL-E 3 | $0.04 (약 52원) | $0.08 (약 104원) | 우수 |

**DALL-E 3는 DALL-E 2보다 2배 비싸지만, 훨씬 더 높은 품질의 이미지를 생성합니다.**

## 일기 이미지 생성 비용 계산

현재 코드는 **DALL-E 3 Standard (1024×1024)**를 사용합니다:

### 월별 예상 비용
- **하루 1장**: $0.04 × 30일 = **$1.20** (약 **1,560원**)
- **하루 1장 + 재생성 1회**: $0.04 × 60장 = **$2.40** (약 **3,120원**)

### 연간 예상 비용
- **하루 1장**: $0.04 × 365일 = **$14.60** (약 **18,980원**)
- **하루 1장 + 재생성 1회**: $0.04 × 730장 = **$29.20** (약 **37,960원**)

## 비용 절감 방법

### 1. DALL-E 2 사용 (50% 절감)
`src/services/aiImageService.js` 파일 수정:
```javascript
// 54번째 줄
model: 'dall-e-2', // dall-e-3 대신 dall-e-2 사용
```

**비용**: $0.02 per image (약 26원)
- 월별: **$0.60** (약 **780원**)
- 연간: **$7.30** (약 **9,490원**)

### 2. 이미지 재생성 최소화
- 이미지가 만족스러우면 재생성하지 않기
- Edge Function을 배포하여 이미지 영구 저장 (재생성 불필요)

### 3. 선택적 이미지 생성
- 특별한 날짜에만 이미지 생성
- 사용자가 원할 때만 생성

## 품질 비교

### DALL-E 2
- ✅ 저렴한 가격
- ✅ 빠른 생성 속도
- ❌ 상대적으로 낮은 품질
- ❌ 프롬프트 이해도 낮음

### DALL-E 3
- ✅ 매우 높은 품질
- ✅ 프롬프트 이해도 우수
- ✅ 더 자연스러운 이미지
- ❌ 2배 비싼 가격

## 추천

### 예산이 제한적인 경우
→ **DALL-E 2 사용** (품질은 약간 낮지만 충분히 사용 가능)

### 품질을 중시하는 경우
→ **DALL-E 3 사용** (현재 설정, 월 $1.20 정도면 합리적)
