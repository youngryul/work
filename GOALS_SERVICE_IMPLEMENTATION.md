## 🚀 사용 방법

### 1. 데이터베이스 설정

Supabase에서 다음 SQL 파일을 실행하세요:

```bash
# Supabase SQL Editor에서 실행
supabase-goals-schema.sql
```

### 2. 앱 실행

```bash
npm run dev
```

### 3. 기능 사용

1. **연간 목표 등록**
   - "목표 관리" 탭 클릭
   - "연간 목표 추가" 버튼 클릭
   - 영역 선택 및 목표 정보 입력
   - 최대 5개까지 등록 가능

2. **월별 목표 등록**
   - "월별 목표" 탭 클릭
   - 월 선택
   - "월별 목표 추가" 버튼 클릭
   - 연간 목표와 연결하여 등록
   - 최대 3개까지 등록 가능
   - ⚠️ 이전 달 회고 작성 필수

3. **일간 체크리스트**
   - 대시보드에서 "오늘의 체크리스트" 확인
   - 주간 행동에서 자동 생성된 항목 체크
   - 수동으로 항목 추가 가능
   - 완료 시 체크박스 클릭

4. **월별 회고**
   - "회고" 탭 클릭
   - 필수 항목 작성 (달성률, 가장 잘한 선택, 실패 원인)
   - 저장 후 다음 달 목표 등록 가능

## 📋 핵심 기능 상세

### 연간 목표 관리
- **제약 조건**:
  - 최대 5개 영역
  - 영역당 1개 목표
  - 영역: 커리어, 건강, 관계, 돈, 성장

- **입력 필드**:
  - 목표 제목 (필수)
  - 목표 설명 (왜 중요한지)
  - 측정 기준 (정량/정성)
  - 방해 요소
  - 대응 전략

### 월별 목표 관리
- **제약 조건**:
  - 최대 3개
  - 반드시 연간 목표와 연결
  - 회고 작성 후 다음 달 목표 등록 가능

- **입력 필드**:
  - 연간 목표 선택 (필수)
  - 목표 제목 (필수)
  - 목표 설명

### 일간 체크리스트
- **자동 생성**:
  - 주간 행동이 있으면 매일 해당 날짜에 자동으로 체크리스트 항목 생성
  - 오늘 할 일에도 자동 추가

- **수동 추가**:
  - 사용자가 직접 항목 추가 가능

### 회고 시스템
- **필수 조건**:
  - 월말 회고 작성 필수
  - 회고 작성 전 다음 달 목표 등록 불가

- **회고 질문**:
  1. 이번 달 목표 달성률 (0-100%)
  2. 가장 잘한 선택 (필수)
  3. 실패 원인 분석 - 의지 vs 환경 (필수)
  4. 다음 달 유지할 것
  5. 다음 달 버릴 것
  6. 자유 회고
